About Alganize

Alganize is a gene expression explorer for algal metabolite pathway genes. It visualizes log2 fold-change data across stress conditions, helping researchers quickly identify regulated genes in pathways like carotenogenesis, fatty acid biosynthesis, and osmoprotectant metabolism.

Data is loaded from meta.json and log2fc.tsv files hosted alongside this page. To add a dataset, place its files in a folder named by GEO accession and register it in registry.json.

Expected file structure
/registry.json
/GSE125419/meta.json
/GSE125419/log2fc.tsv
/GSE92514/meta.json
/GSE92514/log2fc.tsv
